# Traduction Française — *Uma Musume: Pretty Derby*

Bienvenue dans le projet de traduction française de *Uma Musume: Pretty Derby*.  
Ce dépôt a pour objectif de proposer une localisation complète et soignée du jeu afin de le rendre accessible au public francophone.

## Objectif du projet
- Traduire l’ensemble du contenu textuel du jeu (interface, dialogues, compétences, événements, scénarios…)
- Offrir une traduction fidèle à l’œuvre originale tout en restant fluide et naturelle en français
- Mettre à disposition une méthode simple pour installer la traduction

## État actuel
Le projet est encore en développement.  
Certains éléments peuvent être incomplets ou sujets à modification.
